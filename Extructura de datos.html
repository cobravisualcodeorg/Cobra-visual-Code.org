Estructuras de Datos en Python

Listas

Las listas son uno de los tipos de datos más utilizados en Python. Una lista es una colección ordenada de elementos que pueden ser de diferentes tipos (números, cadenas, listas, etc.). Las listas son mutables, lo que significa que puedes modificar sus elementos.

Creación de una lista

Para crear una lista, se usan corchetes [] y los elementos dentro de la lista se separan por comas.

# Ejemplo de lista
frutas = ["manzana", "plátano", "cereza"]
print(frutas)
Explicación: Este ejemplo crea una lista llamada frutas con tres elementos y luego imprime la lista completa.

Acceder a elementos de una lista

Los elementos de una lista se acceden mediante índices, comenzando desde 0. El índice de un elemento es su posición dentro de la lista.

# Accediendo a un elemento de la lista
print(frutas[0])  # Imprime "manzana"
Explicación: El índice 0 accede al primer elemento de la lista, en este caso "manzana".

Modificar elementos en una lista

Puedes modificar los elementos de una lista accediendo al índice correspondiente y asignando un nuevo valor.

# Modificando un elemento de la lista
frutas[1] = "naranja"
print(frutas)
Explicación: En este caso, el elemento en el índice 1 ("plátano") se reemplaza por "naranja".

Agregar y eliminar elementos de una lista

Puedes agregar nuevos elementos a una lista usando el método append(), y eliminar elementos usando remove() o pop().

# Agregando un elemento
frutas.append("kiwi")
print(frutas)

# Eliminando un elemento
frutas.remove("cereza")
print(frutas)
Explicación: Se agrega un "kiwi" al final de la lista y luego se elimina "cereza".

Tuplas

Las tuplas son muy similares a las listas, pero son inmutables. Esto significa que una vez que se crea una tupla, no puedes modificar sus elementos.

Creación de una tupla

Una tupla se crea utilizando paréntesis () y sus elementos se separan por comas.

# Ejemplo de tupla
colores = ("rojo", "verde", "azul")
print(colores)
Explicación: En este caso, creamos una tupla llamada colores que contiene tres colores.

Accediendo a los elementos de una tupla

Al igual que con las listas, puedes acceder a los elementos de una tupla mediante su índice.

# Accediendo a un elemento de la tupla
print(colores[1])  # Imprime "verde"
Explicación: El índice 1 accede al segundo elemento de la tupla, "verde".

Diccionarios

Un diccionario es una colección desordenada de pares clave-valor. Cada elemento tiene una clave única, que se utiliza para acceder al valor asociado.

Creación de un diccionario

Un diccionario se crea utilizando llaves {}, y los elementos se definen como clave: valor.

# Ejemplo de diccionario
persona = {"nombre": "Juan", "edad": 30, "ciudad": "Madrid"}
print(persona)
Explicación: En este caso, el diccionario persona contiene tres pares clave-valor: nombre, edad y ciudad.

Acceder a los valores de un diccionario

Para acceder a los valores de un diccionario, se utiliza la clave correspondiente.

# Accediendo a un valor del diccionario
print(persona["nombre"])  # Imprime "Juan"
Explicación: La clave "nombre" accede al valor asociado, en este caso, "Juan".

Agregar y eliminar elementos en un diccionario

Puedes agregar nuevos pares clave-valor utilizando la clave y asignando un valor. Para eliminar un elemento, puedes usar el método pop().

# Agregando un elemento
persona["profesión"] = "Ingeniero"
print(persona)

# Eliminando un elemento
persona.pop("edad")
print(persona)
Explicación: Se agrega un nuevo par clave-valor "profesión: Ingeniero" y luego se elimina la clave "edad".

Conjuntos

Los conjuntos son colecciones desordenadas de elementos únicos. Los conjuntos no permiten duplicados y no garantizan el orden de los elementos.

Creación de un conjunto

Un conjunto se crea utilizando la función set() o con llaves.

# Ejemplo de conjunto
frutas_set = {"manzana", "plátano", "cereza"}
print(frutas_set)
Explicación: Este ejemplo crea un conjunto llamado frutas_set con tres elementos.
