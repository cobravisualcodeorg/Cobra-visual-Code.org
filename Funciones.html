Funciones en Python

¿Qué son las funciones?

Las funciones en Python son bloques de código reutilizables que se pueden definir para ejecutar tareas específicas. Ayudan a organizar el código, reducir la repetición y mejorar la legibilidad del programa.

Definición de una función

Para definir una función en Python, usamos la palabra clave def seguida del nombre de la función y paréntesis. Luego, dentro de los dos puntos, colocamos el código que se ejecutará cuando se llame a la función.

# Ejemplo de definición de una función
def saludar():
    print("¡Hola, bienvenido a Python!")
Explicación: Este es un ejemplo simple de una función llamada saludar que imprime un mensaje en pantalla cuando se ejecuta.

Cómo llamar a una función

Una vez que hemos definido una función, podemos llamarla por su nombre seguido de paréntesis. Es importante recordar que la función no se ejecutará hasta que la llamamos explícitamente.

# Llamada a la función
saludar()  # Esto ejecutará la función y mostrará el mensaje
Explicación: La llamada a saludar() ejecutará el código dentro de la función, mostrando el mensaje "¡Hola, bienvenido a Python!".

Funciones con parámetros

Las funciones pueden recibir datos de entrada a través de parámetros. Estos parámetros se definen entre los paréntesis al momento de declarar la función.

# Ejemplo de función con parámetros
def saludar_persona(nombre):
    print(f"¡Hola, {nombre}!")
    
# Llamada a la función con un argumento
saludar_persona("Carlos")
Explicación: La función saludar_persona toma un parámetro llamado nombre y lo utiliza para personalizar el saludo.

Funciones con valor de retorno

Las funciones también pueden devolver un valor usando la palabra clave return. El valor devuelto se puede usar más adelante en el programa.

# Ejemplo de función con retorno
def sumar(a, b):
    return a + b

resultado = sumar(5, 3)
print(resultado)  # Esto imprimirá 8
Explicación: La función sumar recibe dos parámetros, realiza una operación y devuelve el resultado. El resultado es almacenado en la variable resultado.

Funciones con múltiples valores de retorno

Una función también puede devolver múltiples valores a la vez, usando tuplas. Estos valores se pueden asignar a diferentes variables al llamar a la función.

# Ejemplo de múltiples valores de retorno
def operaciones(a, b):
    suma = a + b
    resta = a - b
    return suma, resta

resultado_suma, resultado_resta = operaciones(10, 5)
print(resultado_suma)  # Imprime 15
print(resultado_resta)  # Imprime 5
Explicación: La función operaciones devuelve dos valores: la suma y la resta de los dos números proporcionados. Estos valores se asignan a dos variables diferentes.

Funciones anidadas

En Python, también podemos definir funciones dentro de otras funciones. Esto se llama funciones anidadas y es útil en situaciones donde necesitas una función que se utilice solo dentro de otra función.

# Ejemplo de función anidada
def exterior():
    def interior():
        print("Esto es una función anidada.")
    interior()

exterior()  # Llamará a la función exterior que a su vez llamará a la función interior
Explicación: La función exterior llama a la función interior desde su interior. Esto demuestra cómo se puede trabajar con funciones dentro de otras funciones.
