¿Qué es Python?

Python es un lenguaje de programación interpretado, de alto nivel y de propósito general. Fue creado por Guido van Rossum y lanzado en 1991. Su diseño enfatiza la legibilidad del código, lo que facilita su aprendizaje para los principiantes y mejora la productividad de los programadores.

Python es conocido por ser sencillo y muy versátil. Se utiliza en desarrollo web, análisis de datos, inteligencia artificial, automatización de tareas, y mucho más.

¿Por qué elegir Python?

Algunas de las principales ventajas de Python son:

  • Fácil de aprender: Tiene una sintaxis clara y fácil de entender, lo que lo hace ideal para principiantes.
  • Comunidad activa: Hay una enorme comunidad de Python, lo que garantiza que siempre haya recursos, ayuda y nuevas librerías.
  • Versatilidad: Desde desarrollo web (con Django, Flask) hasta análisis de datos (con Pandas, NumPy), Python es perfecto para cualquier tipo de proyecto.
  • Gran cantidad de bibliotecas: Python tiene una enorme colección de bibliotecas que facilitan realizar tareas complejas con pocas líneas de código.

Instalación de Python

Para instalar Python en tu computadora, sigue estos pasos:

  1. Ve a la página oficial de Python.org.
  2. Selecciona la versión adecuada para tu sistema operativo (Windows, macOS, Linux).
  3. Descarga el instalador y ejecútalo. Asegúrate de seleccionar la opción "Add Python to PATH" durante la instalación.
  4. Una vez instalado, abre la terminal o línea de comandos y escribe python --version para asegurarte de que Python se instaló correctamente.

Tu Primer Programa en Python

Vamos a escribir el clásico "¡Hola, Mundo!" en Python.

# Este es el primer programa en Python
print("¡Hola, Mundo!")

Este código imprime el mensaje "¡Hola, Mundo!" en la consola.

Guarda este código en un archivo llamado hola_mundo.py y ejecútalo desde la terminal:

python hola_mundo.py

Conceptos Básicos de Python

A continuación, explicaremos algunos conceptos clave para que puedas empezar a escribir más programas.

Variables y Tipos de Datos

En Python, las variables se usan para almacenar datos. Python es un lenguaje con tipado dinámico, lo que significa que no necesitas declarar el tipo de datos al crear una variable.

# Ejemplo de variables
nombre = "Juan"
edad = 25
altura = 1.75

En este ejemplo, hemos creado tres variables: nombre, edad y altura, con tipos de datos diferentes:

  • nombre: una cadena de texto (string).
  • edad: un número entero (integer).
  • altura: un número flotante (float).

Operaciones Básicas

Python permite realizar operaciones matemáticas fácilmente. Por ejemplo:

# Operaciones en Python
a = 10
b = 5
suma = a + b
resta = a - b
multiplicacion = a * b
division = a / b

Estas son algunas operaciones básicas en Python:

  • a + b: suma.
  • a - b: resta.
  • a * b: multiplicación.
  • a / b: división.
