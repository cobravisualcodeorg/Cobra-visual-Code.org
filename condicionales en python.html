Condicionales en Python

Los condicionales en Python nos permiten ejecutar bloques de código dependiendo de si se cumple o no una condición. Usamos las instrucciones if, elif y else para crear estas decisiones lógicas.

La estructura básica de una sentencia condicional es:

# Estructura básica de un condicional
if condición:
    # Código si la condición es verdadera
elif otra_condición:
    # Código si la otra condición es verdadera
else:
    # Código si ninguna condición se cumple

Ejemplo: Verificar si un número es mayor que 10

Vamos a usar un condicional para verificar si un número es mayor que 10:

numero = 15
if numero > 10:
    print("El número es mayor que 10.")
else:
    print("El número no es mayor que 10.")

En este caso, como el número es 15, el programa imprimirá "El número es mayor que 10."

Condicionales anidados

Podemos también crear condicionales dentro de otros condicionales para hacer decisiones más complejas:

edad = 20
if edad >= 18:
    print("Eres adulto.")
    if edad >= 65:
        print("También eres un adulto mayor.")
else:
    print("Eres menor de edad.")

Bucles en Python

Los bucles nos permiten ejecutar un bloque de código varias veces. En Python, tenemos dos tipos de bucles: for y while.

Bucle for

El bucle for se usa generalmente para iterar sobre una secuencia (como una lista o un rango de números). Aquí tienes un ejemplo que imprime los números del 1 al 5:

for i in range(1, 6):
    print(i)

El range(1, 6) genera una secuencia de números desde 1 hasta 5. El bucle for imprimirá cada número en esa secuencia.

Bucle while

El bucle while repite un bloque de código mientras se cumpla una condición. Aquí tienes un ejemplo de cómo usarlo:

contador = 0
while contador < 5:
    print(contador)
    contador += 1

En este ejemplo, el bucle continuará ejecutándose hasta que la variable contador sea igual a 5.

Break y Continue

Dentro de los bucles, podemos usar las instrucciones break y continue para controlar el flujo del programa:

# Usando break
for i in range(1, 10):
    if i == 5:
        break  # Sale del bucle cuando i es 5
    print(i)

# Usando continue
for i in range(1, 10):
    if i == 5:
        continue  # Salta a la siguiente iteración cuando i es 5
    print(i)

Funciones en Python

Las funciones son bloques de código reutilizables que realizan una tarea específica. Se definen con la palabra clave def.

Definir una función

Para definir una función en Python, usamos la palabra clave def, seguida de un nombre para la función y paréntesis. Por ejemplo:

def saludo():
    print("¡Hola, Mundo!")

# Llamada a la función
saludo()

Este código define una función llamada saludo, que imprime un mensaje en la consola. Luego, se llama a la función con saludo().

Funciones con parámetros

Las funciones también pueden recibir parámetros para personalizar su comportamiento:

def saludo_personalizado(nombre):
    print(f"¡Hola, {nombre}!")

# Llamada a la función con un parámetro
saludo_personalizado("Juan")

En este caso, la función saludo_personalizado toma un parámetro nombre, y luego imprime un mensaje personalizado.

Funciones con valor de retorno

Las funciones también pueden devolver valores utilizando la palabra clave return:

def suma(a, b):
    return a + b

resultado = suma(10, 5)
print(f"El resultado de la suma es: {resultado}")

Este código define una función suma que toma dos parámetros y devuelve su suma. Luego, el resultado se almacena en la variable resultado y se imprime en la consola.
