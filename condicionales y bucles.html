Condicionales y Bucles en Python

Condicionales: if, elif, else

En Python, los condicionales permiten tomar decisiones en función de una condición. Se utilizan las palabras clave if, elif y else.

Uso de if

El condicional if evalúa una expresión y ejecuta un bloque de código si la expresión es verdadera.

# Ejemplo de condicional if
edad = 18
if edad >= 18:
    print("Eres mayor de edad")
Explicación: En este caso, si la variable edad es mayor o igual a 18, se imprime el mensaje "Eres mayor de edad".

Uso de elif y else

El bloque elif (abreviatura de "else if") se usa cuando queremos comprobar varias condiciones. El bloque else se ejecuta si ninguna de las condiciones anteriores es verdadera.

# Ejemplo de condicional elif y else
edad = 15
if edad >= 18:
    print("Eres mayor de edad")
elif edad >= 13:
    print("Eres un adolescente")
else:
    print("Eres un niño")
Explicación: En este ejemplo, si la variable edad es mayor o igual a 18, se imprime "Eres mayor de edad". Si no, se comprueba si es mayor o igual a 13, en cuyo caso se imprime "Eres un adolescente". Si ninguna condición es verdadera, se imprime "Eres un niño".

Bucles: for y while

Los bucles nos permiten ejecutar un bloque de código repetidamente. Los dos tipos de bucles más comunes son el bucle for y el bucle while.

Bucle for

El bucle for se utiliza para iterar sobre una secuencia (como una lista, una cadena de texto, o un rango de números).

# Ejemplo de bucle for
for i in range(5):
    print(i)
Explicación: En este ejemplo, el bucle for recorre los números del 0 al 4 (el rango generado por range(5)) y los imprime uno a uno.

Bucle while

El bucle while se ejecuta mientras una condición sea verdadera. La condición se evalúa antes de cada iteración.

# Ejemplo de bucle while
contador = 0
while contador < 5:
    print(contador)
    contador += 1
Explicación: En este caso, el bucle while continúa ejecutándose mientras contador sea menor que 5. En cada iteración, se incrementa el valor de contador.

Uso de break y continue

Dentro de los bucles, puedes usar break para salir del bucle y continue para saltar a la siguiente iteración del bucle.

# Ejemplo de break y continue
for i in range(5):
    if i == 3:
        break  # Sale del bucle cuando i es 3
    if i == 1:
        continue  # Salta a la siguiente iteración cuando i es 1
    print(i)
Explicación: En este ejemplo, el bucle imprime 0, 2, y luego sale del bucle cuando i es igual a 3 debido al break. El continue hace que se salte la iteración cuando i es igual a 1.
