Condicionales y Operadores Lógicos en Python

¿Qué son los condicionales?

Los condicionales permiten que tu programa ejecute bloques de código específicos según si se cumple o no una condición. Es una de las herramientas más fundamentales en cualquier lenguaje de programación, y en Python se manejan de manera sencilla pero poderosa.

En Python, utilizamos las instrucciones if, else y elif para controlar el flujo del programa según las condiciones que definamos.

La instrucción if

La instrucción if evalúa una condición, y si esa condición se evalúa como verdadera, ejecuta el bloque de código dentro del bloque de la instrucción.

# Ejemplo de la instrucción if
edad = 20
if edad >= 18:
    print("Eres mayor de edad.")
Explicación: Si la variable edad es mayor o igual a 18, el programa imprimirá "Eres mayor de edad.".

La instrucción else

La instrucción else se utiliza cuando la condición de un if no se cumple, y define qué hacer en ese caso. Si la condición es falsa, el bloque de código dentro de else se ejecutará.

# Ejemplo de la instrucción else
edad = 15
if edad >= 18:
    print("Eres mayor de edad.")
else:
    print("Eres menor de edad.")
Explicación: En este caso, dado que la variable edad es menor que 18, se imprimirá "Eres menor de edad.".

La instrucción elif

elif se utiliza cuando queremos comprobar múltiples condiciones. Si la condición del if es falsa, Python evaluará las condiciones definidas con elif.

# Ejemplo con elif
edad = 70
if edad < 18:
    print("Eres menor de edad.")
elif edad >= 18 and edad <= 60:
    print("Eres adulto.")
else:
    print("Eres adulto mayor.")
Explicación: Dependiendo de la edad, el programa clasificará a la persona como menor de edad, adulto o adulto mayor.

Operadores Lógicos en Python

Los operadores lógicos son herramientas muy útiles cuando necesitas combinar múltiples condiciones. En Python, los principales operadores lógicos son:

  • and: Devuelve True si ambas condiciones son verdaderas.
  • or: Devuelve True si al menos una de las condiciones es verdadera.
  • not: Invierte el valor de la condición (si es True, se convierte en False, y viceversa).

Uso del operador and

El operador and se usa para combinar dos o más condiciones. La expresión completa será verdadera solo si todas las condiciones son verdaderas.

# Ejemplo de uso de and
edad = 25
ciudad = "Madrid"
if edad >= 18 and ciudad == "Madrid":
    print("Eres mayor de edad y vives en Madrid.")
Explicación: Ambas condiciones deben ser verdaderas para que el mensaje se imprima.

Uso del operador or

El operador or se usa cuando quieres que se cumpla al menos una de las condiciones. Si cualquiera de las condiciones es verdadera, toda la expresión se evaluará como True.

# Ejemplo de uso de or
edad = 17
ciudad = "Barcelona"
if edad >= 18 or ciudad == "Barcelona":
    print("Eres mayor de edad o vives en Barcelona.")
Explicación: Si la edad es mayor o igual a 18, o si la ciudad es Barcelona, el mensaje se imprimirá.

Uso del operador not

El operador not invierte el valor de una condición. Si una condición es True, el operador not la convierte en False, y viceversa.

# Ejemplo de uso de not
edad = 15
if not edad >= 18:
    print("Eres menor de edad.")
Explicación: Si la condición edad >= 18 es falsa, se imprime el mensaje.

Combinando Condicionales y Operadores Lógicos

Las instrucciones condicionales y los operadores lógicos se pueden combinar de manera muy flexible para crear programas más complejos. Puedes usar if, elif, else junto con and, or, y not para evaluar condiciones más avanzadas.

# Ejemplo avanzado combinando condicionales y operadores lógicos
edad = 25
ciudad = "Madrid"
trabajo = "Programador"
if edad >= 18 and (ciudad == "Madrid" or trabajo == "Programador"):
    print("Eres un adulto que vive en Madrid o es programador.")
else:
    print("No cumples las condiciones.")
Explicación: El bloque de código dentro de if se ejecutará si la persona es adulta y vive en Madrid o si es programador.
