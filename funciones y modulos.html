Funciones y Módulos en Python

Funciones en Python

Las funciones en Python son bloques de código reutilizables que realizan una tarea específica. Puedes definir tus propias funciones para organizar mejor tu código y hacer que sea más modular y legible.

Definir una Función

Para definir una función en Python, usamos la palabra clave def, seguida del nombre de la función y unos paréntesis. Dentro de los paréntesis, puedes definir los parámetros que recibirá la función.

# Ejemplo de función simple
def saludo(nombre):
    print(f"Hola, {nombre}!")

# Llamando a la función
saludo("Carlos")
Explicación: En este ejemplo, la función saludo() toma un argumento nombre y imprime un saludo personalizado. Al llamar a la función con saludo("Carlos"), el programa imprime "Hola, Carlos!".

Funciones con Valor de Retorno

Las funciones también pueden devolver un valor utilizando la palabra clave return. Esto permite que la función calcule o procese algo y devuelva el resultado para que pueda ser usado más tarde.

# Ejemplo de función con valor de retorno
def sumar(a, b):
    return a + b

# Llamando a la función
resultado = sumar(3, 4)
print(resultado)
Explicación: En este caso, la función sumar() toma dos argumentos y devuelve la suma de ambos. Luego, guardamos el valor devuelto en la variable resultado y lo imprimimos.

Módulos en Python

Los módulos son archivos de Python que contienen funciones, variables y clases que puedes importar en tu programa para reutilizar código. Python viene con muchos módulos estándar, pero también puedes crear tus propios módulos.

Importar Módulos

Para importar un módulo en Python, utilizamos la palabra clave import.

# Ejemplo de importar un módulo
import math

# Usando una función del módulo math
resultado = math.sqrt(25)
print(resultado)
Explicación: En este ejemplo, importamos el módulo math y usamos su función sqrt() para calcular la raíz cuadrada de 25, que es 5.

Crear tu Propio Módulo

Puedes crear tu propio módulo guardando tus funciones en un archivo de Python. Para usar el módulo, solo tienes que importarlo en el programa donde lo necesites.

# Ejemplo de archivo módulo.py
def multiplicar(a, b):
    return a * b

# Ejemplo en otro archivo principal.py
import modulo

resultado = modulo.multiplicar(3, 4)
print(resultado)
Explicación: En este ejemplo, creamos un módulo llamado modulo.py que contiene una función multiplicar(). Luego, en otro archivo, importamos este módulo y usamos la función para multiplicar dos números.
