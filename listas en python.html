Listas en Python

Las listas en Python son colecciones ordenadas y mutables, lo que significa que puedes cambiar sus elementos después de que se han creado. Son ideales cuando necesitas almacenar una secuencia de elementos.

Las listas se definen usando corchetes [ ] y los elementos dentro de la lista están separados por comas.

# Crear una lista
frutas = ["manzana", "plátano", "cereza"]

# Acceder a un elemento
print(frutas[1])  # Imprime "plátano"

# Modificar un elemento
frutas[0] = "pera"
print(frutas)  # Imprime ["pera", "plátano", "cereza"]

# Añadir un nuevo elemento
frutas.append("uva")
print(frutas)  # Imprime ["pera", "plátano", "cereza", "uva"]

# Eliminar un elemento
frutas.remove("plátano")
print(frutas)  # Imprime ["pera", "cereza", "uva"]

Las listas permiten diversas operaciones como añadir, eliminar y modificar elementos fácilmente.

Diccionarios en Python

Los diccionarios en Python son colecciones desordenadas de pares clave-valor. A diferencia de las listas, no mantienen un orden específico, pero permiten un acceso rápido a los elementos a través de claves únicas.

Se definen utilizando llaves { } y cada par clave-valor está separado por dos puntos :.

# Crear un diccionario
persona = {
    "nombre": "Juan",
    "edad": 30,
    "ocupación": "desarrollador"
}

# Acceder a un valor por su clave
print(persona["nombre"])  # Imprime "Juan"

# Añadir un nuevo par clave-valor
persona["ciudad"] = "Madrid"
print(persona)

# Modificar un valor existente
persona["edad"] = 31
print(persona)

# Eliminar un par clave-valor
del persona["ocupación"]
print(persona)

Los diccionarios permiten almacenar datos relacionados, lo que los hace muy útiles cuando necesitamos organizar información más compleja.

Tuplas en Python

Las tuplas en Python son similares a las listas, pero a diferencia de las listas, son inmutables, lo que significa que no puedes cambiar sus elementos una vez que han sido creadas.

Se definen utilizando paréntesis ( ), y sus elementos también están separados por comas.

# Crear una tupla
colores = ("rojo", "verde", "azul")

# Acceder a un elemento
print(colores[1])  # Imprime "verde"

# Las tuplas son inmutables, por lo tanto, no puedes cambiar un valor después de la creación
# colores[0] = "amarillo"  # Esto dará un error

Las tuplas son útiles cuando quieres asegurarte de que los datos no cambien durante la ejecución del programa, y son generalmente más rápidas que las listas cuando se utilizan en grandes cantidades de datos.
