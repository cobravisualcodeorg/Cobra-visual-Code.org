Manejo de Archivos y Excepciones en Python

Manejo de Archivos

En Python, trabajar con archivos es una tarea común que se puede hacer fácilmente usando funciones integradas. Los archivos pueden ser leídos, escritos o incluso abiertos en diferentes modos, según las necesidades del programa.

Abrir un archivo

Para abrir un archivo en Python, usamos la función open(), pasando el nombre del archivo y el modo en el que queremos trabajar con él. Los modos más comunes son:

  • r: Leer (modo predeterminado, si no se especifica).
  • w: Escribir (crea un nuevo archivo o sobrescribe uno existente).
  • a: Añadir (agrega contenido al final del archivo).
# Ejemplo de apertura de un archivo en modo lectura
archivo = open("mi_archivo.txt", "r")
contenido = archivo.read()
print(contenido)
archivo.close()
Explicación: Este código abre un archivo llamado "mi_archivo.txt" en modo de lectura, lee su contenido y luego cierra el archivo.

Escribir en un archivo

Si queremos escribir en un archivo, podemos usar el modo w o a. Si el archivo no existe, Python lo creará.

# Ejemplo de escritura en un archivo
archivo = open("mi_archivo.txt", "w")
archivo.write("Este es un nuevo contenido.")
archivo.close()
Explicación: Este código crea o sobrescribe el archivo "mi_archivo.txt" con el texto especificado.

Excepciones en Python

Las excepciones son errores que pueden ocurrir durante la ejecución de un programa. Python ofrece una forma de manejar estos errores para evitar que el programa se detenga inesperadamente.

Uso de try y except

La estructura básica de manejo de excepciones en Python es el bloque try-except. Se coloca el código que puede generar un error dentro del bloque try, y si ocurre una excepción, se maneja con el bloque except.

# Ejemplo de manejo de excepciones
try:
    numero = int(input("Ingresa un número: "))
    print(10 / numero)
except ZeroDivisionError:
    print("No se puede dividir por cero.")
except ValueError:
    print("Debes ingresar un número válido.")
Explicación: Este código maneja dos tipos de excepciones: división por cero y entrada de datos no numéricos.

El bloque finally

El bloque finally se ejecuta independientemente de si ocurrió una excepción o no. Se utiliza para realizar tareas de limpieza, como cerrar archivos o liberar recursos.

# Ejemplo con finally
try:
    archivo = open("mi_archivo.txt", "r")
    contenido = archivo.read()
except FileNotFoundError:
    print("El archivo no se encontró.")
finally:
    archivo.close()
Explicación: Incluso si ocurre un error al intentar leer el archivo, el bloque finally garantiza que el archivo se cierre correctamente.
