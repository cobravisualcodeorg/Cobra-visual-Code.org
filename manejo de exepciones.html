Proyecto Final: Calculadora en Python

Ahora vamos a poner en práctica todo lo aprendido con un mini-proyecto. Vamos a crear una calculadora en Python que permitirá sumar, restar, multiplicar y dividir números. Además, esta calculadora manejará excepciones si el usuario ingresa valores no válidos.

Código de la calculadora:


# Calculadora en Python

# Función para sumar dos números
def sumar(x, y):
    return x + y

# Función para restar dos números
def restar(x, y):
    return x - y

# Función para multiplicar dos números
def multiplicar(x, y):
    return x * y

# Función para dividir dos números con manejo de excepciones
def dividir(x, y):
    try:
        return x / y
    except ZeroDivisionError:  # Si se intenta dividir por cero
        return "Error: No se puede dividir entre cero"

# Menú de opciones para el usuario
def menu():
    print("Calculadora en Python")
    print("Selecciona una operación:")
    print("1. Sumar")
    print("2. Restar")
    print("3. Multiplicar")
    print("4. Dividir")
    print("5. Salir")

# Función principal para ejecutar la calculadora
def calculadora():
    while True:
        menu()  # Mostrar el menú
        opcion = input("Ingresa el número de la operación que deseas realizar: ")

        # Si el usuario elige salir
        if opcion == '5':
            print("¡Gracias por usar la calculadora!")
            break

        # Verificar que la opción es válida
        if opcion not in ['1', '2', '3', '4']:
            print("Opción no válida. Intenta de nuevo.")
            continue

        try:
            num1 = float(input("Ingresa el primer número: "))
            num2 = float(input("Ingresa el segundo número: "))
        except ValueError:
            print("Error: Por favor ingresa un número válido.")
            continue

        # Realizar la operación según la opción
        if opcion == '1':
            print(f"{num1} + {num2} = {sumar(num1, num2)}")
        elif opcion == '2':
            print(f"{num1} - {num2} = {restar(num1, num2)}")
        elif opcion == '3':
            print(f"{num1} * {num2} = {multiplicar(num1, num2)}")
        elif opcion == '4':
            print(f"{num1} / {num2} = {dividir(num1, num2)}")

# Ejecutar la calculadora
calculadora()

Explicación del código:

  • Las funciones sumar, restar, multiplicar y dividir realizan las operaciones matemáticas básicas.
  • El manejo de excepciones en la función dividir permite manejar el error de división por cero.
  • El bucle while permite que el programa siga ejecutándose hasta que el usuario decida salir.
  • El código también verifica si los valores ingresados son números válidos usando un bloque try-except.
